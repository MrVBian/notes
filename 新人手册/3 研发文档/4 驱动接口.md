## 2.1 move指令

- 函数接口

>  void moveXXX(string robotId, string ip, double[] link);

- 请求参数

| 请求参数 | 参数类型  | 参数说明          |
| :------- | :-------- | :---------------- |
| robotId  | string    | Builder内机器人Id |
| ip       | string    | 机器人Ip          |
| link     | double[7] | 机器人关节状态值  |

- 返回参数

| 返回参数 | 参数类型 | 参数说明                        |
| :------- | :------- | :------------------------------ |
| True     | bool     | Trus：下发成功  False：下发异常 |
